# Paq Packages(2)

## Usage

1. Install [paq](https://github.com/Saverio976/paq?tab=readme-ov-file#install)

2. Add this repo

```bash
paq config add-repo "https://github.com/Saverio976/paq-packages2/releases/latest/download/paq-packages.toml"
```
